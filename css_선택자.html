선택자의종류

html파일을 css를 통해 꾸밀수 있고 꾸미기 위해서는 지정된 선택을 해야한다.

작성법

선택자 { 속성명 : 속성값; }

선택자종류 작성방법 설명
전체선택자 * HTML 내부의 모든 태그를 불러서 설정한다.
태그선택자 태그명 HTNL 내부의 불려진 모든 태그를 설정
ID선택자 #아이디이름 속성값 ID의 이름을 불러서 특정ID만 설정
클래스선택자 .클래스이름 속성값 class의 이름을 불러서 특정 클래스만 설정
id와 class의 차이점:
아이디는 한페이지에 한번만 사용(아이디의 이름이 하나라는 뜻), 중복사용안됨
한번사용한 아이디이름은 두번사용 안됨
class는 중복사용 가능, 한 클래스에 여러 클래스명 사용가능, 클래스명 한칸띄어쓰기로 여러 클래스명 부여가능
