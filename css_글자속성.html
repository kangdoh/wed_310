글자속성

글자에 관련된 속성은 글꼴,글자크기,글자굵기 기울임,줄간격,대소문자,글자색상 등 주된 일곱 가지를 필수적으로 알고 있어야한다.

color를 제외한 위에 6가지는 한줄쓰기 가능
속성명 종류 설명
font-family 컴퓨터에 등록되어 있는 폰트명/ 웹폰트활용가능 글꼴을 인용부호에 감싼다
다국적사용자를 대상으로 웹페이를 제공할때 serif(명조체),Sans-serif(고딕체) 글꼴 마지막에 작성
font-size 기본16px를 100%기준으로 사용합니다. 수치의 px등의 단위를 붙여서 지정한다.
font-style italic / normal 글자의 기울기를 지정
font-variant small-caps / normal 작은대문자로 변경
font-weight bold / normal / 숫자지정 - 400이 기본값, 600이상 볼드 글자의 두께를 지정한다. font의 종류의 따라 달라지며 일정한 값은 아니다.
line-height 1은 100%의 값으로 줄간격 없음을 표기, 기본값은 1.4(140%) 글자의 높낮이를 따지는 행간을 조정한다.
font라는 속성으로 일열쓰기 가능
font : (font-weight,font-style,font-variant) (font-size/line-height) (font-family)
세가지 순서대로 작성하되, 글자크기와 글꼴을 생략할수 없음
(괄호)안의 내용은 순서가 바뀔수 있고ㅡ 실제 작성시 괄호는 쓰지 않음
font 라는 속성이 우선순위 이며 나중에 위에 이름값과 같지 않더라도 덮어쓰기 가능

글자관련추가속성

주로글자나 단어보다는 문장에서 사용하는 속성

  • text-aline
    • 특징 - 왼쪽,오른쪽, 가운데, 양쪽정렬을 나타냄
    • 주의사항 - 블럭요소에서만 사용가능
    • justify는 내용안에 줄바꿈이 일어날만큼 양이 있을경우 보임
  • text-indent
    • 특징 - 문단의 첫머리 들여쓰기 기능
    • 주의사항 - 블럭만 사용 / 단위작성
  • text-decoration
    • underline - 밑줄
    • linethrough - 가운데줄(취소선)
    • overline - 윗줄
    • none - 제거
  • letter-spacing - 자간설정 단위적용
  • text-transform
    • uppercase-모두대문자변경
    • lowercase-소문자로변경
    • capitalize-첫글자만 대문자(띄어쓰기 후 대문자)
